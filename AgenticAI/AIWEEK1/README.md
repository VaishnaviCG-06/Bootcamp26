# 🤖 SmartAssist AI

SmartAssist AI is a multi-tool AI web application built using **React**, **Vite**, and the **Google Gemini API**. It provides different AI-powered features to help users learn, study, and improve their writing.

## ✨ Features

* 📘 **Explain a Concept** – Understand topics with simple explanations and examples.
* 🧸 **Explain to a Toddler** – Get child-friendly explanations with easy language.
* 📝 **Summarize Text** – Convert long text into short, clear summaries.
* 🎯 **Generate Quiz** – Create multiple-choice quiz questions from any topic.
* ✍️ **Professional Writer** – Rewrite text in a professional tone.
* 📚 **Flashcards** – Generate study flashcards for quick revision.

## 🛠️ Technologies Used

* React
* Vite
* Google Gemini API
* React Markdown
* Remark GFM
* CSS

## 🚀 How to Run the Project

1. Clone the repository:

   ```
   git clone <repository-link>
   ```

2. Install dependencies:

   ```
   npm install
   ```

3. Create a `.env` file and add your Gemini API key:

   ```
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. Start the development server:

   ```
   npm run dev
   ```

5. Open the local URL shown in the terminal (usually `http://localhost:5173`).
