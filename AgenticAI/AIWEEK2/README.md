# 🎓 Prompt Studio

Prompt Studio is an AI-powered web application built using **React**, **Vite**, and the **Google Gemini API**. It provides multiple prompt modes designed to assist with learning, writing, interview preparation, and productivity.

## ✨ Features

* 🧒 **Simple Explainer** – Explains difficult topics in an easy-to-understand way.
* 💼 **Career Polisher** – Rewrites text professionally for workplace communication.
* 📝 **Quick Notes** – Summarizes long text into concise key points.
* 🧪 **Practice Quiz** – Generates multiple-choice questions for self-assessment.
* 🎯 **Mentor** – Provides guidance
